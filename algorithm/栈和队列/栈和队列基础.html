# 👉 栈和队列基础

# 栈 - 后进先出，只用 push 和 pop 完成增删的“数组”

栈，后进先出，只允许从尾部操作，从尾部增加/删除，因此只能通过 push / pop 完成增删。

栈元素出栈时，要取的是栈顶元素，栈顶元素指的就是数组尾部的元素。

# 队列 - 先进先出，只用 push 和 shift 完成增删的“数组”

栈，先进先出，只允许从尾部增加和从头部移除，因此只能通过 push / shift 完成增删。

队列元素出队时，要取的是队头元素，队头元素指的是数组的第一个元素。
